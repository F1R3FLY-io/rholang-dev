# Community Events

Join the F1r3fly Rholang community at conferences, meetups, and online events.

## Upcoming Events

### 💻 Monthly Developer Calls
**When**: First Thursday of each month, 5:00 PM UTC  
**Where**: [Discord](https://discord.gg/nCQxEwUd)  
**Topics**:
- Recent developments
- Community Q&A
- Technical discussions

## Regular Meetups

### Online Meetups

#### Rholang Study Group
- **Frequency**: Weekly, Wednesdays 6:00 PM UTC
- **Platform**: Discord Voice Channel
- **Focus**: Learning Rholang together, code reviews, pair programming

#### Office Hours with Core Devs
- **Frequency**: Bi-weekly, Fridays 4:00 PM UTC
- **Platform**: Zoom
- **Purpose**: Get help with technical questions directly from core developers

### Regional Meetups

Regional meetups are forming based on interest. If you would like to host a local gathering, post in the F1r3fly Discord to coordinate dates and content.
